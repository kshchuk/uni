# Management information systems — data analytics track

Practical analytics labs in **Python** (Jupyter): **NumPy**, **Pandas**, data cleaning, visualisation, and progressively richer analysis on real-world tabular datasets (e.g. IMDb-style movie data). Later folders extend the same theme with structured exercises and optional ML-style tasks.

## Labs

| --- | --- |

| [Lab-2.ipynb](./Lab-2.ipynb) | Lab 2 — data cleaning and basic visualisation. |
| [Lab-3.ipynb](./Lab-3.ipynb) | Lab 3 — further analysis (course-specific tasks in the notebook). |
| [Lab-4.ipynb](./Lab-4.ipynb) | Lab 4 — continued analytics workflow. |
