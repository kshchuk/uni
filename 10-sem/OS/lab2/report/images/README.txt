Drop screenshots here with the following filenames. The report will pick
them up automatically (and fall back to placeholder frames otherwise):

  main-window.png         Main application window on launch.
  png-preview.png         A .png file selected: IHDR info + image preview.
  jpeg-meta.png           A .jpg/.jpeg file selected: metadata block.
  folder-count.png        A folder selected: jpg/png count in the folder.
  parent-navigation.png   After pressing "Parent folder" to go up.

Any PNG/JPG will do. The macros in report.tex use \includegraphics and
scale to the text width, so regular screenshots look fine.
