# Лабораторний проєкт 4 — Клієнт–серверне сортування масиву

**Курс:** Операційні системи  
**Мова:** Go 1.22  
**Сортування:** QuickSort  
**Розмір масиву:** 10 000 `int32`  
**Експериментів:** 100  

## Постановка

Клієнт формує масив, надсилає на сервер по TCP; сервер сортує і повертає результат. Клієнт вимірює час від початку відправлення до завершення читання відповіді.

## Протокол

`uint32 N` (LE) + `N × int32` (LE). Round-trip: 80 008 байт (запит + відповідь).

## Результати (macOS loopback, seed=42)

## Запуск

```bash
make build
make run-server   # термінал 1
make run-client   # термінал 2
# або
make experiment
```
