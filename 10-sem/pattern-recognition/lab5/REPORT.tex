\section{Мета та завдання роботи}

\subsection{Мета}
Для номера у списку $N=2$ реалізувати VAR 3 (3 класи, 3D), причому вибір проєкції виконати за трьома критеріями, як вказано у методичці.

\subsection{Визначення варіанту}
\[
N \bmod 3 = 2 \Rightarrow \text{VAR 3}.
\]

\subsection{Завдання}
\begin{enumerate}
    \item Згенерувати 3 навчальні класи у $\mathbb{R}^3$ (рівномірні розподіли по координатах).
    \item Обчислити центри мас класів.
    \item Побудувати 3 прямі проєктування: 1-2, 2-3, 1-3.
    \item Обчислити 3 критерії для кожної прямої і обрати найкращу.
    \item Проєктувати тестові точки на обрану пряму та класифікувати за найближчою проєкцією центра.
\end{enumerate}

%================ ТЕОРЕТИЧНІ ОСНОВИ ================
\section{Метод}

Нехай для прямої $L$ (з напрямом $u$ і опорною точкою $o$) проєкція точки $x$:
\[
t(x)=(x-o)^\top u.
\]

Для кожного класу $k$ маємо множину проєкцій $\{t_{k,i}\}$.
Використані критерії:
\begin{align}
J_1(L) &= \sum_{k=1}^{3}\left(\max_i t_{k,i} - \min_i t_{k,i}\right) \quad (\min), \\
J_2(L) &= \sum_{i<j}|\bar t_i - \bar t_j| \quad (\max), \\
J_3(L) &= \sum_{k=1}^{3}\mathrm{Var}(t_k) \quad (\min).
\end{align}

Після min-max нормалізації:
\[
S(L)=\frac{(1-\widetilde J_1)+\widetilde J_2+(1-\widetilde J_3)}{3},
\]
обираємо $L^*=\arg\max S(L)$.

Класифікація точки $x$:
\[
\hat y(x)=\arg\min_k |t(x)-t(\mu_k)|,
\]
де $\mu_k$ --- центр класу.

def criteria(line):
    pj, mu = {}, {}
    for cn in class_names:
        t = proj(train[cn], line)
        pj[cn] = t
        mu[cn] = float(t.mean())

    j1 = sum(float(t.max() - t.min()) for t in pj.values())
    m = [mu[n] for n in class_names]
    j2 = abs(m[0]-m[1]) + abs(m[0]-m[2]) + abs(m[1]-m[2])
    j3 = sum(float(np.var(t)) for t in pj.values())
    return j1, j2, j3, pj, mu
\end{lstlisting}
