\section{Мета та завдання роботи}

\subsection{Мета}
Класифікувати тестову точку в $\mathbb{R}^4$ за трьома класами, використовуючи два способи побудови \emph{середнього представника} класу та два правила порівняння з тестом.

\subsection{Завдання}
\begin{enumerate}
    \item Задати навчальні множини трьох класів (по три 4-вимірні вектори).
    \item Обчислити представників класів: середина діапазону по вимірах і арифметичне середнє.
    \item Обчислити ваги $w_j$ через сумарний розмах $S_j$ по всіх класах.
    \item Класифікувати тестову точку $\mathbf{t}=(6,6,26,5)$ правилом голосування та зваженою сумою модулів.
    \item Порівняти результати для обох підходів; оформити таблиці та ілюстрації.
\end{enumerate}

\section{Теоретичні відомості}

Нехай клас $C_k$ містить точки $\mathbf{x}^{(i)}\in\mathbb{R}^d$.

\textbf{Підхід 1} (середина діапазону):
\begin{equation}
  \mu_{k,j}=\tfrac{1}{2}\Bigl(\min_{\mathbf{x}\in C_k} x_j + \max_{\mathbf{x}\in C_k} x_j\Bigr).
\end{equation}

\textbf{Підхід 2} (середнє арифметичне):
\begin{equation}
  \boldsymbol{\mu}_k = \frac{1}{|C_k|}\sum_{\mathbf{x}\in C_k}\mathbf{x}.
\end{equation}

Розмах класу по виміру $j$: $s_{k,j}=\max x_j-\min x_j$ у $C_k$. Сумарний розмах $S_j=\sum_k s_{k,j}$, вага $w_j=1/S_j$.

\textbf{Голосування:} для кожного $j$ клас з $\min_k |t_j-\mu_{k,j}|$ отримує один голос; переможець --- $\arg\max_k$ голосів.

\textbf{Зважена відстань:}
\begin{equation}
  D_k(\mathbf{t})=\sum_{j=1}^{d} w_j\,|t_j-\mu_{k,j}|,\qquad
  \hat{k}=\arg\min_k D_k(\mathbf{t}).
\end{equation}

\subsection{Голосування}

Для \textbf{підходу 1} найближчі представники по вимірах: $x_1$ і $x_2$ --- Синя, $x_3$ і $x_4$ --- Червона. На вимірі $x_2$ відстані до Синьої та Червоної однакові ($|6-6|=0$), тому обидва класи отримують голос. Підсумок: Синя~2, Червона~3 --- перемагає \textbf{Червона}.

Для \textbf{підходу 2} представники Синьої та Червоної зміщуються ($x_1$ Синьої $\approx 6{,}33$), але розподіл голосів той самий: Червона перемагає з рахунком 3:2.

Для червоного класу при підході~1 координата $y$ представника: $(\min+\max)/2=(5+7)/2=6$ (навчальні $y\in\{7,6,5\}$).

\section{Висновки}

\begin{enumerate}
    \item Для тестової точки $(6,6,26,5)$ обидва правила зваженої суми відносять її до \textbf{Червоного} класу при обох підходах до представника.
    \item Голосування також дає \textbf{Червону} (3:2 проти Синьої); зелений клас не отримує жодного голосу, бо представник $(2,5,5,8)$ далекий по $x_1$ та $x_3$.
    \item Підхід~2 змінює лише представник Синьої по $x_1$ та $x_4$, тому голосування не змінює переможця; $D_k$ злегка зменшуються, але порядок класів той самий.
    \item Зважування $w_j=1/S_j$ узгоджує виміри з різним масштабом ($x_1$ має великий розмах); без ваг домінував би вимір з найбільшими абсолютними координатами.
\end{enumerate}
