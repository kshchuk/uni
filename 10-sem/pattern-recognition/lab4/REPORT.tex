\section{Мета та завдання роботи}

\subsection{Мета}
Класифікувати тестову точку в $\mathbb{R}^4$ за трьома класами, використовуючи два способи побудови \emph{середнього представника} класу та два правила порівняння з тестом.

\subsection{Завдання}
\begin{enumerate}
    \item Задати навчальні множини трьох класів (по три 4-вимірні вектори).
    \item Обчислити представників класів: середина діапазону по вимірах і арифметичне середнє.
    \item Обчислити ваги $w_j$ через сумарний розмах $S_j$ по всіх класах.
    \item Класифікувати тестову точку $\mathbf{t}=(6,6,26,5)$ правилом голосування та зваженою сумою модулів.
    \item Порівняти результати для обох підходів; оформити таблиці та ілюстрації.
\end{enumerate}

\section{Теоретичні відомості}

Нехай клас $C_k$ містить точки $\mathbf{x}^{(i)}\in\mathbb{R}^d$.

\textbf{Підхід 1} (середина діапазону):
\begin{equation}
  \mu_{k,j}=\tfrac{1}{2}\Bigl(\min_{\mathbf{x}\in C_k} x_j + \max_{\mathbf{x}\in C_k} x_j\Bigr).
\end{equation}

\textbf{Підхід 2} (середнє арифметичне):
\begin{equation}
  \boldsymbol{\mu}_k = \frac{1}{|C_k|}\sum_{\mathbf{x}\in C_k}\mathbf{x}.
\end{equation}

Розмах класу по виміру $j$: $s_{k,j}=\max x_j-\min x_j$ у $C_k$. Сумарний розмах $S_j=\sum_k s_{k,j}$, вага $w_j=1/S_j$.

\textbf{Голосування:} для кожного $j$ клас з $\min_k |t_j-\mu_{k,j}|$ отримує один голос; переможець --- $\arg\max_k$ голосів.

\textbf{Зважена відстань:}
\begin{equation}
  D_k(\mathbf{t})=\sum_{j=1}^{d} w_j\,|t_j-\mu_{k,j}|,\qquad
  \hat{k}=\arg\min_k D_k(\mathbf{t}).
\end{equation}

\subsection{Порівняння чотирьох варіантів}

Алгоритм має чотири комбінації: два способи побудови представника (підхід~1 і~2) та два правила класифікації (голосування і зважена сума). Зведення --- у табл.~\ref{tab:lab4matrix}; деталізація голосування по вимірах --- у табл.~\ref{tab:lab4voterange}--\ref{tab:lab4votemean}, значення $D_k$ --- у табл.~\ref{tab:lab4weighted}.

\textbf{Голосування.} Для обох підходів переможець --- \textbf{Червона} з рахунком 0:2:3. На $x_2$ відстані до Синьої та Червоної однакові ($|6-6|=0$), тому обидва класи отримують голос на цьому вимірі. Підхід~2 зміщує представник Синьої ($x_1\approx 6{,}33$), але розподіл голосів не змінюється.

\textbf{Зважена сума.} Мінімальне $D_k$ для обох підходів --- у \textbf{Червоного} класу ($D_{\min}=1{,}0170$ і $1{,}0417$). Внесок $x_3$ найбільший: $|26-29{,}5|$ або $|26-29{,}67|$ менші за відстані до зеленого та синього представників.

Для червоного класу при підході~1 координата $x_2$ представника: $(\min+\max)/2=(5+7)/2=6$ (навчальні $x_2\in\{7,6,5\}$).

\section{Висновки}

\begin{enumerate}
    \item Усі \textbf{чотири варіанти} алгоритму (табл.~\ref{tab:lab4matrix}) відносять тестову точку $(6,6,26,5)$ до \textbf{Червоного} класу.
    \item Голосування: рахунок 0:2:3; зелений клас не отримує голосів, бо представник $(2,5,5,8)$ далекий по $x_1$ та $x_3$.
    \item Зважена сума: $D_k$ мінімальне у червоного; підхід~2 лише злегка змінює $D_k$, порядок класів той самий.
    \item Зважування $w_j=1/S_j$ узгоджує виміри з різним масштабом; без ваг домінував би вимір з найбільшими абсолютними координатами.
\end{enumerate}
