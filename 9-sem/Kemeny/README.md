# Kemeny's constant — seminar presentation

Beamer slides (Ukrainian, with English references) on **Kemeny's constant** for finite ergodic Markov chains: intuition via “time to equilibrium”, connections to mean first-passage times, and modern matrix formulations (Doyle 2005; Bini–Hunter–Latouche–Meini–Taylor 2018).
